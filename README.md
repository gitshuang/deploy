## 将单个html文件 进行部署
`docker build -t name . `
+ .是docker文件所在目录
  
`docker run --rm -p 80:3000 name`
+ 80是容器端口,3000 是宿主机端口
## Dockerfile
`ADD index.html /usr/share/nginx/html/`
+ 是将可静态文件放在ngnix 静态文件目录

+ 镜像内nginx默认配置路径是/etc/nginx/conf.d/default.conf
